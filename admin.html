Option recommandée — télécharger :
  1. Clique « Télécharger le fichier » ci-dessus. Ça enregistre formations-data.js dans ton dossier Téléchargements.
  2. Déplace ce fichier dans assets/js/ de ton dossier lvlup-site, en écrasant l'ancien (l'Explorateur te demandera de confirmer le remplacement — accepte).
  3. Dans le terminal, dans le dossier du projet : git add -A && git commit -m "Mise à jour catalogue" && git push
Option copier-coller (si tu préfères) : clique « Copier le code », ouvre assets/js/formations-data.js dans VS Code, sélectionne tout (Ctrl+A), colle (Ctrl+V), sauvegarde. Le fichier s'appelle bien formations-data.js — il y a aussi un formations-render.js dans le même dossier, celui-là ne se touche jamais.

Tu peux aussi coller ce code directement dans le chat à Claude et lui demander de sauvegarder et pousser pour toi.
